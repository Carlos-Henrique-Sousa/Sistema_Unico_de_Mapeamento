Autenticação

A API utiliza JWT (JSON Web Tokens) para autenticação. Inclua o token no header Authorization: Bearer <token>

curl -H "Authorization: Bearer YOUR_JWT_TOKEN" https://api.sum.local/api/endpoint

Visão Geral

A API do S.U.M fornece endpoints para gerenciar escolas, estudantes, professores, atividades e o sistema de mapeamento inteligente de salas de aula. 
# Base URL
https://api.sum.local/api/

# Exemplo de requisição
curl -X GET https://api.sum.local/api/health/

Autenticação

Endpoints para autenticação e gerenciamento de usuários.
POST /api/auth/login/ Login de usuário 
{
  "identifier": "usuario123",
  "password": "senha123"
}

Escolas

Gerenciamento de escolas e turmas.
GET /api/escola/ Listar escolas
POST /api/escola/ Criar escola
GET /api/escola/turmas/ Listar turmas

Estudantes

Gerenciamento de estudantes e suas informações acadêmicas.
GET /api/estudantes/ Listar estudantes
GET /api/estudantes/{id}/ Detalhes do estudante
GET /api/estudantes/{id}/notas/ Notas do estudante

Professores

Gerenciamento de professores e suas turmas.
GET /api/professores/ Listar professores
GET /api/professores/{id}/turmas/ Turmas do professor

Atividades

Sistema de atividades e avaliações.
GET /api/atividades/ Listar atividades
POST /api/atividades/ Criar atividade
GET /api/atividades/{id}/respostas/ Respostas da atividade

Mapeamento

Sistema de mapeamento inteligente de salas de aula.
POST /api/mapeamento/gerar/ Gerar novo mapeamento
GET /api/mapeamento/atual/{uuid}/ Mapeamento atual
POST /api/mapeamento/mover/ Mover aluno
GET /api/mapeamento/historico/{turma_id}/ Histórico de mapeamentos

Eventos

Sistema de eventos e agenda escolar.
GET /api/eventos/ Listar eventos
POST /api/eventos/ Criar evento
