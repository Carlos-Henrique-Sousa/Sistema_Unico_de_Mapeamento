Autenticação

A API utiliza JWT (JSON Web Tokens) para autenticação. Inclua o token no header Authorization: Bearer <token>

curl -H "Authorization: Bearer YOUR_JWT_TOKEN" https://api.sum.local/api/endpoint

Visão Geral

A API do S.U.M fornece endpoints para gerenciar escolas, estudantes, professores, atividades e o sistema de mapeamento inteligente de salas de aula.

# Base URL https://api.sum.local/api/ # Exemplo de requisição curl -X GET https://api.sum.local/api/health/

Autenticação

Endpoints para autenticação e gerenciamento de usuários.

POST /api/auth/login/ Login de usuário
{ "identifier": "usuario123", "password": "senha123" }

Resposta de Sucesso:

{ "access": "eyJ0eXAiOiJKV1QiLCJhbGc...", "refresh": "eyJ0eXAiOiJKV1QiLCJh...", "user": { "id": 1, "identifier": "usuario123", "nome": "João Silva" } }
200 OK 401 Unauthorized

Escolas

Gerenciamento de escolas e turmas.

GET /api/escola/ Listar escolas
POST /api/escola/ Criar escola
GET /api/escola/turmas/ Listar turmas

Mapeamento

Sistema de mapeamento inteligente de salas de aula.

POST /api/mapeamento/gerar/ Gerar novo mapeamento
GET /api/mapeamento/atual/{uuid}/ Mapeamento atual
